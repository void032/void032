## ⚡ universal shader studio &nbsp;·&nbsp; [live](https://shader-studio-teal.vercel.app) &nbsp;·&nbsp; [source](https://github.com/void032/shader-studio)

> browser-based glsl editor — write shaders live on terrain, primitives, or any `.glb` model. export as three.js shadermaterial, r3f component, raw glsl, or vanilla webgl. no install, no backend.

[![universal shader studio](https://github.com/void032/shader-studio/raw/main/preview.png)](https://shader-studio-teal.vercel.app)

`three.js r168` &nbsp; `react 18` &nbsp; `glsl` &nbsp; `vite` &nbsp; `perlin noise` &nbsp; `gltfloader`

- 🏔️ **procedural terrain** — perlin & island noise, configurable resolution / height / octaves / frequency
- 📦 **glb drag-drop** — drag any `.glb` into the scene, auto-centered and scaled, geometry merged
- ⚡ **live glsl editor** — vertex + fragment with syntax highlighting and built-in shader presets
- 📤 **multi-format export** — three.js shadermaterial · r3f component · raw glsl · vanilla webgl boilerplate

**j3dp — cinematic 3d portfolio**

scroll-driven camera traveling a spline through six atmospheric scenes. custom shadermaterial sky dome on an inverted sphere, sphere-cluster clouds with fresnel edge transparency, `mountain.glb` with layered depth planes, gsap scrolltrigger-driven camera, fog/smoke transitions. no react, no gc in the raf loop.
